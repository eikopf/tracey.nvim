# Todo Items

## CRUD Operations

r[todo.create]
Users can create a new todo item with a title and optional description.

r[todo.read]
Users can retrieve a single todo item by its ID.

r[todo.list]
Users can list all todo items, optionally filtered by completion status.

r[todo.update]
Users can update the title, description, or completion status of a todo item.

r[todo.delete]
Users can delete a todo item by its ID.

## Validation

r[todo.validation.title]
Todo titles must be between 1 and 200 characters.

r[todo.validation.description]
Todo descriptions must be at most 2000 characters.
